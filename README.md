# NEXA Admin Portal v2.1

## Setup

### Backend
```bash
cd backend
pip install -r requirements.txt
# Edit .env with your MongoDB URI
python app.py
# Runs on http://localhost:5000
```

### Frontend
```bash
cd frontend
npm install
npm run dev
# Runs on http://localhost:5173
```

## Features
- Dashboard, Staff Profiles, Certifications, Project Status, AI Allocation, Staff ID
- Certifications: View, Analyze (PDF authenticity), Approve, Reject
- Staff ID: Generate / Re-Generate → creates Employee Portal login automatically
- Staff Profiles: Approved certifications auto-stored, skills auto-added

## Default Admin Login
- Email: any email
- Password: any password (demo mode when no users exist)

Register at /create-account to create a real admin account.
